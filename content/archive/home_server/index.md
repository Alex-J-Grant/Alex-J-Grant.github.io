### Overview 
This is a general cyber security project that I decided to start for fun. Its main use is as a testing area for me to test things
and as a learning tool for security regarding servers.

### Current Features
- SSH Hardened
- Firewall
- Automatic backups every night


### Things I learned Through this
- How to use and secure SSH
- Networking
- Using IP Tables command to create a firewall
- Using cronjobs to backup the servers data automatically
